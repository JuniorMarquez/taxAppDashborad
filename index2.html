Arquitectura

Diagrama 1

  • Base de Datos NoSQL en servidor cloud, sincronizable con Base de Datos NoSQL en cliente.

  • API REST FULL : Rutinas en servidor para atención y respuesta de solicitudes.

  • WebApp : Aplicación web de una sola página (SPA) para la administración de la plataforma con un (1) nivel de acceso (administrador).

  • PWA : Aplicación Web Progresiva, aplicación de una sola página con dos (2) niveles de acceso (taxista, pasajero) y características de aplicaciones nativas

    • App Shel

    • WebSockets

    • Realtime (trabajo en tiempo real)

    • Agregar al escritorio
